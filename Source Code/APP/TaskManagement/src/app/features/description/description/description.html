Project Description

This is a simple Task Management System developed using Angular and the .NET Core Web API.

Features
  • Create new users.
  • Basic authentication for API endpoints.
  • RESTful APIs for creating, editing, viewing, and deleting tasks, as well as marking tasks as completed.
  • Error handling and validation.
  • Sorting and filtering of the task list.
Tools Used
  • SQL Server Management Studio 2022
  • Visual Studio Code 2025
  • Visual Studio 2022
Technologies Used
  • Angular, TypeScript, Bootstrap, HTML, CSS, JavaScript
  • ASP.NET Core Web API, Entity Framework Core
Configuration Notes

Before running the application, ensure the following:

  1. Angular Application
    • Update the API URL in environment.development.ts
  2. ASP.NET Core Web API
    • Configure CORS in Program.cs
    • Update database connection string in appsettings.json
    • Verify the database context configuration in Program.cs
Other Notes
  • All users' full names appear in the task assignment dropdown.
